
El funcionamiento de la computación cuántica consiste en manipular un estado cuántico mediante una serie de puertas cuánticas, después de un tiempo se realiza una medición para obtener el resultado. **Si la computadora cuántica consta de un solo cubit, cuando se haga la medición el resultado sera un bit clásico y si es de varios cubits entonces la medición sera de varios bits clásicos.**

Supongamos que un cubit esta en el estado $\large \alpha \ket 0 + \beta \ket 1$ cuando medimos este cubit en la base computacional, lo que se obtiene es un bit de información clásica $\large 0$ con probabilidad de $\large |\alpha|^2$ y $\large 1$ con probalidad de $\large |\beta|^2$, porque recordemos que [[Amplitudes|las sumas de las amplitudes deben ser igual a 1.]]. Después el resultado se convierte directamente en un estado de base computacional como lo es $\large \ket 0$ o **$\large \ket 1$.

Ademas, una vez que se haya hecho la medición independientemente del resultado $\huge \alpha$ y $\large \beta$ se "van". es decir, que ya no hay forma de saber información sobre ellos, se convierten en un especie de información oculta.
