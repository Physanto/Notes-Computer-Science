# configuración de la red en linux

La configuración de red que se va a realizar esta enfocada para conexiones a red por cable ethernet

Esta configuración de red básica e inicial es para conexiones a la red por medio de cable Ethernet ademas sirve para cualquier distribución basada en linux ya que 

la herramienta que se va a usar es moderna y estándar del paquete [***iproute2***](https://wiki.linuxfoundation.org/networking/iproute2).

## ip

Es una utilidad de sistema que hace parte del paquete iproute2 esta nos sirve para la administración de redes y control del trafico en linux.
